# LLMOps

An LLM application operations platform built with Flask and LangChain.

## Architecture

The project adopts a 7-layer architecture with clear separation of concerns.

| Layer | Modules |
|-------|---------|
| **Presentation** | PC Website / Open API / Mobile App |
| **Gateway** | Nginx / Load Balancer |
| **Control** | AI App Module / User Module / Auth Module / Plugin Module |
| **Service** | LLM Service / Upload Service / OAuth Service / Embedding Service |
| **Core** | Multi-LLM Integration / Agent Orchestration / LangChain Wrapper |
| **Storage** | PostgreSQL / Redis / Vector DB / Object Storage |
| **Infrastructure** | Linux / Docker / Kubernetes |

**Presentation Layer**: End-user entry points — PC website, open API, and mobile app.

**Gateway Layer**: Nginx receives all external traffic and distributes requests via load balancing.

**Control Layer**: Core business logic — AI app management, user system, authorization, and plugin extensions.

**Service Layer**: Concrete functional services — LLM invocation, file upload, auth validation, and text vectorization.

**Core Layer**: Encapsulates low-level AI capabilities — multiple LLM providers, Agent orchestration, and the LangChain toolkit.

**Storage Layer**: Data persistence — PostgreSQL for structured data, Redis for caching, vector DB for semantic search, and object storage for files.

**Infrastructure Layer**: The foundation — Linux servers, Docker containers, and Kubernetes for orchestration and autoscaling.

## Getting Started

### Prerequisites

- Python 3.11+
- PostgreSQL 15+
- Redis 7+

### Install Dependencies

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### Configure Environment Variables

Copy `.env` and fill in your values:

```bash
cp .env .env.local
```

```dotenv
FLASK_ENV=development
DATABASE_URL=postgresql://user:password@localhost:5432/llmops
REDIS_URL=redis://localhost:6379/0
SECRET_KEY=your-secret-key
JWT_SECRET_KEY=your-jwt-secret
GITHUB_CLIENT_ID=your-github-client-id
GITHUB_CLIENT_SECRET=your-github-client-secret
```

### Initialize the Database

```bash
flask db init
flask db migrate -m "init"
flask db upgrade
```

### Run the Server

```bash
flask run
# or
python app.py
```

The server runs at `http://localhost:5000` by default.
